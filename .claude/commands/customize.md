# /project:customize

テンプレートファイルをカスタマイズするためのインタラクティブコマンドです。

## 実行内容

以下のテンプレートファイルの一覧を表示し、ユーザーが選択したファイルを編集できるようにします。

```
カスタマイズ可能なテンプレート:

1. templates/slides/theme.css
   → スライドの配色・フォント・レイアウトを変更

2. templates/index-template.md
   → 目次ページのレイアウト・セクション構成を変更

3. templates/knowledge-asset.yaml
   → メタ情報のスキーマ・拡張フィールドを変更

番号を選択してください（複数可、カンマ区切り）:
```

ユーザーの選択に応じて:

1. 選択されたファイルの現在の内容を表示
2. 「何を変更したいですか？」と質問
3. ユーザーの指示に従って Edit ツールで修正
4. 変更後の内容をプレビュー表示
5. 「この変更でよいですか？(OK/やり直し)」と確認

## カスタマイズ例

### テーマカラーの変更
```
ユーザー: メインカラーを青から緑に変えたい
→ theme.css の #1E2761 を #2E7D32 に一括置換
```

### 目次セクションの並び替え
```
ユーザー: スライドセクションを一番上に持ってきたい
→ index-template.md のセクション順序を変更
```

### メタ情報フィールドの追加
```
ユーザー: チーム名フィールドを追加したい
→ knowledge-asset.yaml に team フィールドを追加
```
